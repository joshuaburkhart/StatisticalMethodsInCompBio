#HW 1 Data Overview#

Number of Tuples: 768
Number of Attributes: 8 plus class 

Attributes (all numeric)
   1. Number of pregnancies
   2. Glucose tolerance (Plasma glucose concentration @ 2 hrs)
   3. Diastolic blood pressure (mm Hg)
   4. Body Fat: Triceps skin fold thickness (mm)
   5. Insulin: 2-Hour serum insulin (mu U/ml)
   6. Body mass index (weight in kg/(height in m)^2)
   7. Expected Genetic Influence of affected and unaffected relatives on subject's eventual risk
   8. Age (years)
   9. Class variable (0 or 1)

Missing Attribute Values: None

Class Distribution: (class value 1 is interpreted as "tested positive for
   diabetes")
